# IME Cursor Indicator

A lightweight utility that displays a small indicator near your cursor when IME is ON.

## Features

- Shows a dot next to the cursor while IME is active
- Works globally across all windows
- Runs in the system tray
- Customizable indicator colors
- Minimal performance impact

## Usage

1. Run the executable: IMECursorIndicator.exe
2. The app runs in the system tray.
3. Right-click the tray icon to:
   - Change indicator color
   - Exit the application

## Startup (Auto Launch)

To start the app automatically with Windows:

1. Press: Win + R
2. Enter:shell:startup
3. Place a shortcut of the executable in the opened folder
